# Privacy Policy Repository

This repository serves as the central hub for managing and hosting privacy policies for all mobile applications published by **Indus Analytics** on the Google Play Store.

## Purpose

Each app published on the Play Store is required to provide a publicly accessible privacy policy that clearly explains how user data is collected, used, and protected. This repository fulfills that requirement by hosting individual, app-specific privacy policy pages using GitHub Pages.

Having all policies in one place ensures:
- **Consistency** — All apps follow the same structure and standards for privacy disclosures.
- **Maintainability** — Policies can be reviewed and updated from a single repository.
- **Transparency** — Users can easily access and understand how their data is handled for each app.
